# Onyx
Onyx is a CLI game made for simplicity, not fancy graphics. The hope for Onyx is to produce a high quality game without taking up much space or having modern graphics.

# What to Expect
1. This project may have long periods of inactivity. I will not always be here to confirm proposed changes.
2. There may be unexpected major changes. I will accept most changes as long as they are appropriate changes that adhere to the goals of Onyx.
